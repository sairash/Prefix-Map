# Segmap (Segmented Maps)
## Prefix Map Golang

<pre>
- Store
  -> Key: data1 value: Value of key1
  -> Key: data2 value: Value of key2
  -> key: data3 value: Value of key3
</pre>

<pre>
- Get
  -> provide key: data 
  -> get value: [Value of key1, Value of key2, Value of key3]
</pre>
